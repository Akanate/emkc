Instructions

You'll get one input, a string with multiple words. Return the longest word in the string. If the input contains multiple words that are the largest length, return a string that contains all of the words.

Inputs

inputString
Single string with multiple words.

Sample Test Cases

Regular
inputString: "run barn yellow barracuda shark fish swim"
Output: "barracuda"
Same Size
inputString: "fishes sam gollum sauron frodo balrog"
Output: "fishes gollum sauron balrog"
