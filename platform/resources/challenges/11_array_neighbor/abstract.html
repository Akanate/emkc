Instructions

You'll get two input values, each will be a comma separated list of numbers. Each value will contain the same amount of numbers. You'll need to merge the two lists by inserting numbers from value2 into value1. Each number from value2 should be inserted after a number that it is larger than (but not larger than another number from value2).

Inputs

value1
First comma separated list of numbers

value2
Second comma separated list of numbers

Sample Test Cases

Short List
Value 1: 9,5
Value 2: 7,10
Output: 9,10,5,7
Large List
Value 1: 34,18,4,102
Value 2: 15,19,120,64
Output: 34,64,18,19,4,15,102,120
